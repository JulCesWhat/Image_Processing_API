# Image_Processing_API

## Steps to run application
1. Clone project
2. Go to main folder and run `npm i`
3. Run `npm run test` to run the tests
4. Run `npm run build` to build the project
5. Run `npm start` to run the application
6. Once application is running open the follwoing url in a broswer `http://localhost:3000/api/images?filename=icelandwaterfall&width=200&height=200`
